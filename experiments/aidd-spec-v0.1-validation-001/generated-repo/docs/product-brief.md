# Product Brief: IssueBrief Lite

## 目的

IssueBrief Liteは、小さな開発タスクをAIへ渡す前に、目的、ユーザー課題、非ゴール、検証コマンド、証跡メモを1画面で整理する静的Webアプリです。

## 対象ユーザー

- AIエージェントへ実装依頼を出す開発者
- タスク依頼の不足をレビューするチームメンバー
- AIDD-Spec v0.1の入力品質を軽量に検証したい実験者

## 主要体験

1. 入力画面でタスク名、ユーザー課題、非ゴール、検証コマンド、証跡メモを書く。
2. レビュー画面で不足しているAIDD-Spec項目を確認する。
3. 必須項目が揃うと「送信準備OK」を確認する。
4. オフライン、タイムアウト、エラー、解析中の状態をUI上のボタンで再現する。

## 差別化したゴール

単なるフォームではなく、AI Task Packetに必要な項目の不足をその場で可視化します。外部送信、ログイン、DB保存を持たないため、手元で安全に入力品質を確認できます。

## 非ゴール

- 外部API送信はしない
- ログインは作らない
- DB保存はしない
- フレームワークやbuild toolは使わない
- 課金機能は作らない

## 成功条件

- 日本語UIで入力とレビューが完結する
- empty/loading/success/error/offline/timeoutを確認できる
- 不足項目がul/liで表示される
- form controlにlabelがある
- 状態メッセージがaria-liveで通知される
- `npm run lint:static` と `npm run test:contract` が成功する
