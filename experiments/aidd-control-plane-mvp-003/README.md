# AIDD Control Plane MVP 003

## 目的

MVP 002で検査できるようになったAIDD-Spec成果物に、実行ログ・スクリーンショット・CI URLを集めるEvidence Collector UIを追加する。

## 今回の範囲

- Evidence Collector Dashboard
- Command Log Collector
- Screenshot Evidence Collector
- CI / Report Link Collector
- Evidence Preview JSON
- readiness score
- 欠落証跡の表示
- サンプル証跡の挿入
- 証跡不足シナリオ

## 非ゴール

- 実ファイルアップロード
- 外部API送信
- DB保存
- ログイン
- GitHub API連携

## 準拠目標

AIDD-Spec v0.1 L3。
